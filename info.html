DropBoard

Data organization app with drag-and-drop interface and database backend.

Language JavaScript (Next.js, React)
Server Next.js dev server
GitHub adrian3/dropboard  (Public)
What it does
  • Drag-and-drop board for organizing data, tasks, and content into columns
  • Built as a reusable core engine (@adrian3/dropboard-core) with a standalone local app shell
  • Persistent storage via a local database — data survives restarts
  • Used as the board engine inside Ade's World Builder
How to use

Start the dev server with npm run dev from the standalone/ folder, then open http://localhost:3107. Drag cards between columns to reorganize. Changes are saved automatically.

Built with
Next.js React framework — app shell and routing
dnd kit Drag-and-drop library — handles card and column reordering
Prisma ORM — local SQLite database for persistent board state
